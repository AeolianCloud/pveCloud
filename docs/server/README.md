# Server 文档总览

本目录维护 `server/` 的项目事实、契约和架构说明。

## 文档职责

- `architecture.md`
  后端整体架构、边界、鉴权和当前阶段范围
- `../security.md`
  跨后端、前端和运维的通用安全基线
- `go-technical.md`
  技术栈、目录结构、命令和运行依赖
- `logging.md`
  运行日志、访问日志、后台操作审计和管理端日志管理边界
- `audit-actions.md`
  后台审计动作目录和当前管理端写接口审计覆盖情况
- `api/`
  当前开放的管理端 API 契约与跨接口约定
- `database/design.md`
  当前数据库契约与表分组
- `jobs.md`
  通用异步任务、Worker、重试和实例生命周期后台任务说明
- `integrations/`
  外部集成历史说明入口

## 建议阅读顺序

按后端任务定位时，建议先按下面顺序建立上下文：
1. `architecture.md`
2. 涉及鉴权、权限、脱敏、限流、审计、上传、敏感配置或安全边界时读 `../security.md`
3. `go-technical.md`
4. 涉及运行日志、访问日志、审计日志或日志管理时读 `logging.md`
5. 涉及后台审计动作、筛选、展示映射或覆盖检查时读 `audit-actions.md`
6. 再按任务进入对应子域：
   - API：`api/`
   - 数据库：`database/design.md`
   - 异步任务与 Worker：`jobs.md`

## 当前实现范围

- 后端当前只把 `/admin-api/*`、公开站点配置接口、用户账号自助接口、支付宝/微信侧用户实名接口、服务器产品目录接口、用户端订单接口、用户端支付接口、用户端实例接口、用户端工单接口、异步任务管理接口和 `/healthz` 作为现行契约
- `cmd/api` 负责 HTTP 服务入口；`cmd/worker` 负责异步任务领取、实例生命周期和通知后台工作
- MariaDB 是基础后台事实来源
- Redis 是运行时基础依赖，用于管理端会话、验证码、短 TTL 状态、限流和缓存

当前除公开站点配置、用户账号自助接口、支付宝/微信侧用户实名接口、服务器产品目录接口、用户端订单、支付、实例、续费订单、工单和管理端异步任务外，不再把钱包、发票、JSAPI/openid、小程序支付和其它业务流视为现行业务契约。实例交付和到期释放只通过 MCP PVE client API 对接上游 PVE 适配服务，不开放通用 PVE 管理平台能力。
