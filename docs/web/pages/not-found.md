# Not Found 页面契约

## 页面定位

`Not Found` 是404页面，对应路由 `/:pathMatch(.*)*`。

404页面是当用户访问不存在的页面时显示的错误页面。

## 行为范围

- 访问不存在的路由时展示404页面。
- 404页面为公开页面，不要求登录态。
- 404页面展示友好的错误提示和导航引导。
- 用户可以通过导航菜单或返回首页按钮继续浏览。

## 展示内容

### 错误提示
- 404错误代码
- 友好的错误描述："页面未找到"或类似文案
- 可能的原因说明

### 导航引导
- 返回首页按钮：`/`
- 返回上一页按钮（如果可用）
- 导航菜单：首页、产品、价格等

### 搜索功能
- 搜索框（如果适用）
- 热门链接（如果适用）

## 关联接口

- `GET /api/site-config` - 获取站点配置（用于站点名称和Logo）

具体字段、响应和错误码以 `docs/server/api/` 为准。

用户端只调用 `/api/*`，不得调用 `/admin-api/*`。

## 验收重点

- 访问不存在的路由时正确显示404页面。
- 错误提示友好易懂。
- 返回首页按钮功能正常。
- 导航菜单链接正确。
- 响应式设计适配不同屏幕尺寸。
- 页面性能良好，加载速度快。
