# Products 页面契约

## 页面定位

`Products` 是产品展示页，对应路由 `/products`。

产品展示页展示云服务器产品的详细信息，包括产品分类、套餐详情、配置参数和订单创建引导。

## 行为范围

- 访问 `/products` 时展示产品列表和详情。
- 产品展示页为公开页面，不要求登录态。
- 产品展示页展示所有可用的云服务器产品。
- 产品按销售地域筛选展示，地域来源于 `GET /api/server-catalog` 返回的套餐可用销售地域。
- 每个产品展示基础信息、配置参数、价格和购买状态。
- 用户可以点击产品查看详情或选择套餐。
- 产品详情展示完整配置、价格对比和订单创建引导。
- 点击可购买套餐后展示购买配置确认，不直接静默使用默认地域、系统模板或网络类型创建订单。

## 展示内容

### 产品列表
- 产品分类：入门型、标准型、企业型等。
- 每个产品卡片展示：
  - 产品名称
  - 产品简介
  - 核心配置（CPU、内存、存储）
  - 价格范围（月付、季付、半年付、年付）
  - 推荐标识（如果适用）
  - 售罄状态（如果适用）
  - "查看详情"按钮

### 产品详情
- 产品完整描述
- 详细配置参数：
  - CPU 核心数
  - 内存大小
  - 存储类型和容量
  - 带宽
  - 流量
  - IP 地址
  - 操作系统模板
  - 网络类型
- 价格详情：
  - 月付价格
  - 季付价格
  - 半年付价格
  - 年付价格
  - 折扣信息（如果适用）
- 销售地域
- 系统模板列表
- 网络类型列表
- 购买状态：可购买、暂时售罄、即将上线
- 行动号召按钮：
  - 可购买状态：已登录用户进入订单创建；未登录用户跳转 `/login` 并携带站内 `redirect`
  - 售罄状态："暂时售罄"
  - 即将上线："即将上线"

### 购买配置确认
- 购买配置确认只在用户选择固定套餐后出现，不允许用户修改套餐固定规格。
- 可选择项来自当前套餐在 `GET /api/server-catalog` 中返回的可用配置：
  - 计费周期：`monthly`、`quarterly`、`semi_yearly`、`yearly` 中接口返回的 active 周期
  - 销售地域：当前套餐关联的可见 active 销售地域
  - 系统模板：当前套餐关联的可见 active 系统模板
  - 网络类型：当前套餐关联的可见 active 网络类型
- 每个可选择项只有一个可用值时可以默认选中，但页面仍应展示该项，让用户知道订单将使用什么配置。
- 购买配置确认展示订单金额汇总，金额以后端公开目录和订单创建结果为准。
- 用户可填写备注，备注只作为订单人工处理参考，不影响价格或自动交付。
- 当前阶段不提供自定义 CPU、内存、硬盘、带宽、公网 IP 数量、购买数量或登录密码模式。

### 筛选和排序
- 按销售地域筛选；标签包含“全部”和接口返回的地域名称。
- 选择某个地域后，只展示至少包含该地域可用销售地域的套餐。
- 按价格排序
- 按配置排序
- 搜索功能（如果产品较多）

## 关联接口

- `GET /api/server-catalog` - 获取产品目录
- `GET /api/site-config` - 获取站点配置（用于售罄状态等）
- `POST /api/orders` - 创建订单

具体字段、响应和错误码以 `docs/server/api/` 为准。

用户端只调用 `/api/*`，不得调用 `/admin-api/*`。

## 验收重点

- 产品列表加载正常，展示所有可用产品。
- 销售地域标签来自产品目录接口返回数据，按地域筛选功能正常。
- 产品详情展示完整配置和价格信息。
- 价格展示符合契约：月付、季付、半年付、年付价格。
- 销售地域、系统模板和网络类型正确展示。
- 创建订单前必须允许用户确认并选择计费周期、销售地域、系统模板和网络类型。
- 购买状态正确显示：可购买、售罄、即将上线。
- 行动号召按钮只进入订单创建，不承诺支付、实例开通或资源交付。
- 产品图片和描述加载正常。
- 响应式设计适配不同屏幕尺寸。
- 页面性能良好，加载速度快。
