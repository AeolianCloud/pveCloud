# Products 页面契约

## 页面定位

`Products` 是产品展示页，对应路由 `/products`。

产品展示页展示云服务器产品的详细信息，包括产品分类、套餐详情、配置参数和购买引导。

## 行为范围

- 访问 `/products` 时展示产品列表和详情。
- 产品展示页为公开页面，不要求登录态。
- 产品展示页展示所有可用的云服务器产品。
- 产品按分类组织：入门型、标准型、企业型等。
- 每个产品展示基础信息、配置参数、价格和购买状态。
- 用户可以点击产品查看详情或选择套餐。
- 产品详情展示完整配置、价格对比和购买引导。

## 展示内容

### 产品列表
- 产品分类：入门型、标准型、企业型等。
- 每个产品卡片展示：
  - 产品名称
  - 产品简介
  - 核心配置（CPU、内存、存储）
  - 价格范围（月付、季付、半年付、年付）
  - 推荐标识（如果适用）
  - 售罄状态（如果适用）
  - "查看详情"按钮

### 产品详情
- 产品完整描述
- 详细配置参数：
  - CPU 核心数
  - 内存大小
  - 存储类型和容量
  - 带宽
  - 流量
  - IP 地址
  - 操作系统模板
- 价格详情：
  - 月付价格
  - 季付价格
  - 半年付价格
  - 年付价格
  - 折扣信息（如果适用）
- 销售地域
- 系统模板列表
- 购买状态：可购买、暂时售罄、即将上线
- 行动号召按钮：
  - 可购买状态："登录后查看购买入口"
  - 售罄状态："暂时售罄"
  - 即将上线："即将上线"

### 筛选和排序
- 按分类筛选
- 按价格排序
- 按配置排序
- 搜索功能（如果产品较多）

## 关联接口

- `GET /api/server-catalog` - 获取产品目录
- `GET /api/site-config` - 获取站点配置（用于售罄状态等）

具体字段、响应和错误码以 `docs/server/api/` 为准。

用户端只调用 `/api/*`，不得调用 `/admin-api/*`。

## 验收重点

- 产品列表加载正常，展示所有可用产品。
- 产品分类正确，筛选功能正常。
- 产品详情展示完整配置和价格信息。
- 价格展示符合契约：月付、季付、半年付、年付价格。
- 销售地域和系统模板正确展示。
- 购买状态正确显示：可购买、售罄、即将上线。
- 行动号召按钮文案符合契约限制。
- 产品图片和描述加载正常。
- 响应式设计适配不同屏幕尺寸。
- 页面性能良好，加载速度快。
