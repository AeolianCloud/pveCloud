# Pricing 页面契约

## 页面定位

`Pricing` 是价格页，对应路由 `/pricing`。

价格页展示云服务器产品的详细价格信息，包括套餐对比、价格表和购买引导。

## 行为范围

- 访问 `/pricing` 时展示价格信息。
- 价格页为公开页面，不要求登录态。
- 价格页展示所有可用套餐的价格对比。
- 价格按套餐类型组织：入门型、标准型、企业型等。
- 每个套餐展示详细配置和价格。
- 用户可以比较不同套餐的价格和配置。
- 提供购买引导和行动号召按钮。

## 展示内容

### 价格表
- 套餐分类：入门型、标准型、企业型等。
- 每个套餐卡片展示：
  - 套餐名称
  - 套餐简介
  - 核心配置：
    - CPU 核心数
    - 内存大小
    - 存储类型和容量
    - 带宽
    - 流量
    - IP 地址
  - 价格详情：
    - 月付价格
    - 季付价格
    - 半年付价格
    - 年付价格
    - 折扣信息（如果适用）
  - 推荐标识（如果适用）
  - 售罄状态（如果适用）
  - 行动号召按钮

### 价格对比
- 套餐对比表格，展示不同套餐的配置和价格差异。
- 高亮推荐套餐。
- 显示套餐优势和特点。

### 计费说明
- 计费周期：月付、季付、半年付、年付。
- 价格说明：是否含税、是否有隐藏费用。
- 续费说明：自动续费、手动续费。
- 退款政策（如果适用）。

### 购买引导
- 行动号召按钮：
  - 可购买状态："登录后查看购买入口"
  - 售罄状态："暂时售罄"
  - 即将上线："即将上线"
- 联系销售（如果适用）。
- 常见问题解答。

## 关联接口

- `GET /api/server-catalog` - 获取产品目录和价格信息
- `GET /api/site-config` - 获取站点配置（用于售罄状态等）

具体字段、响应和错误码以 `docs/server/api/` 为准。

用户端只调用 `/api/*`，不得调用 `/admin-api/*`。

## 验收重点

- 价格表加载正常，展示所有可用套餐。
- 价格信息准确：月付、季付、半年付、年付价格。
- 配置信息完整：CPU、内存、存储、带宽、流量、IP。
- 套餐对比功能正常，差异清晰可见。
- 推荐套餐标识正确显示。
- 售罄状态正确显示。
- 行动号召按钮文案符合契约限制。
- 计费说明清晰易懂。
- 响应式设计适配不同屏幕尺寸。
- 页面性能良好，加载速度快。
