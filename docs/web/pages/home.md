# Home 页面契约

## 页面定位

`Home` 是官网首页，对应路由 `/`。

首页是用户访问官网的第一个页面，展示品牌信息、核心产品、价格概览和行动号召。

## 行为范围

- 访问 `/` 时展示首页内容。
- 首页为公开页面，不要求登录态。
- 首页展示站点名称、Logo、导航菜单、英雄区域、产品特色、价格概览、客户评价和页脚。
- 导航菜单包含首页、产品、价格、登录、注册等入口。
- 英雄区域展示品牌标语和主要行动号召按钮。
- 产品特色展示云服务器的核心优势。
- 价格概览展示主要套餐和价格范围。
- 客户评价展示用户反馈（如果可用）。
- 页脚包含版权信息、联系方式和相关链接。

## 展示内容

### 站点信息
- 站点名称和Logo来自 `GET /api/site-config`。
- 站点描述和标语来自站点配置。

### 导航菜单
- 首页：`/`
- 产品：`/products`
- 价格：`/pricing`
- 登录：`/login`
- 注册：`/register`
- 用户中心：`/user`（登录后显示）

### 英雄区域
- 品牌标语：展示云服务器销售系统的核心价值。
- 行动号召按钮：引导用户查看产品或注册账号。
- 背景图片或动画（如果可用）。

### 产品特色
- 高性能：展示服务器性能优势。
- 高可用：展示服务可用性保证。
- 安全可靠：展示安全特性。
- 灵活扩展：展示扩展能力。

### 价格概览
- 展示主要套餐：入门型、标准型、企业型等。
- 展示价格范围：月付、季付、半年付、年付价格。
- 展示推荐套餐标识。
- "查看详情"按钮链接到价格页面。

### 客户评价
- 展示用户评价摘要（如果可用）。
- 评价来源和真实性验证。

### 页脚
- 版权信息：当前年份和站点名称。
- 联系方式：邮箱、电话（如果可用）。
- 相关链接：关于我们、隐私政策、服务条款。
- 社交媒体链接（如果可用）。

## 关联接口

- `GET /api/site-config` - 获取站点配置信息
- `GET /api/server-catalog` - 获取产品目录（用于价格概览）

具体字段、响应和错误码以 `docs/server/api/` 为准。

用户端只调用 `/api/*`，不得调用 `/admin-api/*`。

## 验收重点

- 首页加载正常，展示完整内容。
- 站点名称和Logo正确显示。
- 导航菜单链接正确，登录/注册入口可见。
- 英雄区域展示品牌标语和行动号召按钮。
- 产品特色展示核心优势。
- 价格概览展示主要套餐和价格范围。
- 页脚包含版权信息和相关链接。
- 响应式设计适配不同屏幕尺寸。
- 页面性能良好，加载速度快。
